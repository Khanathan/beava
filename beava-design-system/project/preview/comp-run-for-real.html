Skip the browser — run it for real

The simulator runs in your browser. The real beava runs anywhere.

Everything below works against a local beava with the same wire protocol. One binary, 30-second install, and the curl examples in the rest of this page point at localhost:6400 instead of the simulator.

# docker — pinned tag, no install needed
$ docker run -p 6400:6400 -v $(pwd)/data:/var/lib/beava beava/beava:v0

# homebrew — native binary on macOS / Linux
$ brew install beava && beava serve --data-dir ./beava-data

# verify
$ curl http://localhost:6400/health
{"status":"ok","version":"0.1.0"}
