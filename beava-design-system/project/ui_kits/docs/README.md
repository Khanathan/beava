# Docs UI Kit — beava.dev/docs

Docs site with sidebar navigation, prose, code blocks, callouts.

## Components
- `DocsNav.jsx` — top bar (product nav + docs search)
- `DocsSidebar.jsx` — grouped left sidebar with section headers
- `DocsContent.jsx` — article body with headings, prose, code, callouts
- `DocsTOC.jsx` — right-side table of contents
- `CodeBlock.jsx` — syntax-highlighted code with copy button
- `Callout.jsx` — note / tip / warning

`index.html` shows a realistic docs page on "Rolling counters".
