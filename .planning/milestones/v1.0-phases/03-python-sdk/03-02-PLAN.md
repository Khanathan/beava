<!-- RegisterRequest JSON schema the operators must serialize to -->
From src/server/protocol.rs (RegisterRequest DTO):
```rust
pub struct RegisterRequest {
    pub name: String,         // stream name
    pub key_field: String,    // key field name
    pub features: Vec<FeatureDefRequest>,
}
pub struct FeatureDefRequest {
    pub name: String,                   // feature name (Python attribute name)
    #[serde(rename = "type")]
    pub feature_type: String,           // "count", "sum", "avg", "derive" (+ future: "min", "max", "distinct_count", "last", "lookup")
    pub field: Option<String>,          // required for sum, avg, min, max, distinct_count, last
    pub window: Option<String>,         // required for count, sum, avg, min, max, distinct_count
    pub bucket: Option<String>,         // optional bucket granularity
    pub expr: Option<String>,           // required for derive
    pub optional: Option<bool>,         // optional flag for sum, avg
}
```

Supported server-side types (Phase 2): "count", "sum", "avg", "derive"
Future types (Phase 5): "min", "max", "distinct_count", "last", "lookup"
SDK must define ALL 9 operator classes per SDK-03.

    2. Create `python/tally/_stream.py` with:
       - `class StreamMeta(type):` metaclass that in `__new__`:
         a. Collects OperatorBase instances from all bases (walk reversed MRO for mixin support, skip object)
         b. Collects OperatorBase instances from class body namespace (overrides bases)
         c. Sets `cls._tally_features = features` (dict of name -> OperatorBase)
         d. Sets `cls._tally_key_field = kwargs.get('key')` from class keyword args
         e. Sets `cls._tally_stream_name = name` (class name)

       - `def stream(*, key: str):` decorator function that returns a wrapper calling `StreamMeta(name, bases, namespace, key=key)`. Implementation: return a function that takes the class and recreates it with StreamMeta. Simplest approach: `def stream(*, key: str): def decorator(cls): namespace = {k: v for k, v in cls.__dict__.items() if not k.startswith('__')}; return StreamMeta(cls.__name__, cls.__bases__, namespace, key=key)  return decorator`
       - Validate `key` is non-empty string, raise TypeError if not.
