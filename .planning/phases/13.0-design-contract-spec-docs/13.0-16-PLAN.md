<test_data_strategy>

For each op, generate test data with:

1. **Deterministic seed:** `rng = random.Random(<test-specific-seed>)`. Per-test seed varies (42, 43, 44 etc.) so failures are isolated.
2. **Realistic distribution:** events look like real workload shape — user IDs from a small fixed pool (e.g., `["alice", "bob", "carol", "dave", "eve"]`), values from realistic ranges (e.g., purchase amounts $0.50-$500, page IDs from a known set, lat/lon within a bounding box).
3. **Per-entity expected:** Computed in test setup via Python loop over the same input data:
   ```python
   expected = {entity: 0 for entity in entities}
   for event in pushed_events:
       expected[event["user_id"]] += event["amount"]
   # ...then assert
   for entity, value in expected.items():
       assert app.get("Feature", entity)["sum"] == value
   ```
4. **Cold-start:** every test ends with `assert app.get("Feature", "unknown_entity_xyz") == {}`.
5. **Tolerance for sketches:** HLL n_unique within ±5% of true; quantile within ±2 percentile points; top_k order preserved (top item is always #1, second is always #2, etc.); entropy within ±0.05 nats.
6. **Time-controlled tests:** decay / velocity / windowed ops use a deterministic clock helper (`_helpers.py::FakeClock`) so server `now_ms()` is reproducible. If FakeClock isn't viable in v0 (engine controls time), tests use coarse tolerance bounds.

</test_data_strategy>

Pattern per test (using test_count as canonical example — apply to all 8):
```python
import random
import pytest
import beava as bv
from ._helpers import ENTITIES, _engine_available

pytestmark = pytest.mark.skipif(
    not _engine_available(),
    reason="requires Phase 13.4 engine + Phase 13.5 SDK rewrite",
)


def test_count_per_user_high_volume(app):
    """bv.count: push 1000 events across 5 users; verify per-entity count + cold-start."""

    @bv.event
    class Click:
        user_id: str
        page: str

    @bv.table(key="user_id")
    def UserClicks(clicks):
        return clicks.group_by("user_id").agg(
            click_count=bv.count(window="forever"),
        )

    app.register(Click, UserClicks)

    # Push 1000 events with seeded distribution
    rng = random.Random(42)
    expected = {entity: 0 for entity in ENTITIES}
    for _ in range(1000):
        user = rng.choice(ENTITIES)
        expected[user] += 1
        app.push("Click", {"user_id": user, "page": rng.choice(["/home", "/about", "/contact"])})

    # Verify per-entity counts (sum to 1000)
    assert sum(expected.values()) == 1000
    for entity, exp in expected.items():
        result = app.get("UserClicks", entity)
        assert result == {"click_count": exp}, f"{entity}: expected {exp}, got {result}"

    # Cold-start
    assert app.get("UserClicks", "unknown_xyz") == {}
```
