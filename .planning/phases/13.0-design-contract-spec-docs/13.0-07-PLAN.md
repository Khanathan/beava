| [`bv.last_n`](./last_n.md) | O(n) bounded by required `n` kwarg | Tier 1 |
| [`bv.lag`](./lag.md) | O(n) bounded by required `n` kwarg | Tier 1 |

Three of five (first_n / last_n / lag) require an `n` kwarg per [V0-MEM-GOV-02](../../../.planning/REQUIREMENTS.md). Lifetime mode is allowed for all 5 (memory bound is finite).
