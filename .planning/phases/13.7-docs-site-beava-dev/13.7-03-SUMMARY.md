## Asset chosen

- **Format:** SVG (`beava-website/project/assets/quickstart-demo.svg`).
- **File size:** ~3 KB (well under the 500 KB / 2 MB targets).
- **Tooling used:** hand-rolled SVG terminal-screenshot mockup. The output values shown in the SVG (3 impressions per c1, bid_sum 2.25, bid_mean 0.75; 2 impressions per c2, bid_sum 0.65, bid_mean 0.325) were verified by actually running `python3 examples/python/adtech.py` locally.
- **Why SVG (not GIF/PNG):** sharper than PNG at any zoom; smaller than equivalent GIF; no font embedding needed at deploy time; no asciinema/agg required (those tools were not installed). Per Plan 03 Option C (ImageMagick fallback) — SVG is the strictly better fallback.
