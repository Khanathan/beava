## EnrichFromTableTyped Row Projection Algorithm

```rust
fn enrich_from_row(input: &Row, right: Option<&Row>) -> Row {
    let mut out = Row::zeroed(&enriched_schema);
    // 1. Copy input prefix verbatim.
    out.payload[..input_schema.row_size]
        .copy_from_slice(&input.payload[..input_schema.row_size]);
    out.arena.extend_from_slice(&input.arena);
    // 2. For each projected field, read from right at the right's
    //    schema offset and write to out at the projection's dst_offset.
    //    Missing right → projected fields stay zero (D-C2 null-safe).
    if let Some(right) = right {
        for pf in &projected {
            copy_field_between_rows(right, right_schema, &pf.right_field_name,
                                     &mut out, pf.dst_offset, &enriched_schema);
        }
    }
    out
}
```

`copy_field_between_rows` dispatches on the source field's `FieldTy` and uses the matching typed read/write pair — no allocation for scalars, arena extension for strings/bytes.

## derive_enriched_schema

Output schema = input fields + projected fields appended with packed offsets. Projections are marked `nullable: true` (D-C2 — right-row-missing preserved as zero/empty).

```rust
pub fn derive_enriched_schema(
    input: &RegisteredSchema,
    projections: &[(&str, FieldTy)],
    inline_str_cap: u8,
) -> RegisteredSchema {
    let mut fields = input.fields.clone();
    let mut next_offset = input.row_size;
    for (name, ty) in projections {
        fields.push(FieldSpec {
            name: name.to_string(), ty: *ty, offset: next_offset, nullable: true,
        });
        next_offset += ty.fixed_width(inline_str_cap);
    }
    RegisteredSchema { schema_id: 0, name: format!("{}_enriched", input.name),
                       fields, inline_str_cap, row_size: next_offset }
}
```
