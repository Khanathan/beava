<!-- Migration patterns from RESEARCH.md -->

Old pattern:
```python
import tally as st
@st.stream(key="user_id")
class Transactions:
    tx_count_1h = st.count(window="1h")
```

New pattern:
```python
import tally as tl
from tally import source, dataset, group_by

@source
class RawTransactions:
    pass

@dataset(depends_on=[RawTransactions])
class Transactions:
    features = group_by("user_id").agg(
        tx_count_1h=tl.count(window="1h"),
    )
```

Push target changes: app.push(RawTransactions, {...}) instead of app.push(Transactions, {...})

View pattern -> dataset with derive-only:
```python
@dataset(depends_on=[Transactions])
class UserRisk:
    features = group_by("user_id").agg()
    score = tl.derive("Transactions.tx_count_1h > 10")
```

test_protocol.py fix: `encode_push` was renamed to `encode_push_binary` in Phase 11.
