<naming_decision>
The existing Python SDK at `python/` uses distribution name `tally` (hatchling-built, pure-Python). The PyO3 extension cannot reuse that name — we ship it as a native submodule `tally._native` that lives inside the same installed `tally` package. Top-level `tally/__init__.py` re-exports `Pipeline` and all error types from `_native` so users write `from tally import Pipeline`.

Two install modes:
  - Pure-Python SDK for authoring pipelines (existing `python/` hatch build) — unchanged.
  - Native replica extension (this plan) — built from `python-native/` via maturin. Installed wheel places `_native.cpython-3x-x86_64-linux-gnu.so` inside the `tally/` package alongside the Python files.

To deliver both together, maturin's pyproject is configured with `python-source = "../python/tally"` so the wheel bundles the existing Python modules AND the compiled extension. For v0 demo, users pip-install the maturin-built wheel; the hatch wheel remains the "dev author" install path. This is documented in `python-native/README.md`.

Alternative considered: separate `tally_replica` distribution. Rejected — two imports in user code is bad DX for a demo.
</naming_decision>

    ```rust
    #[pyclass(module = "tally._native")]
    pub struct Pipeline {
        config: PipelineConfig,
        session: Option<tally::client::Session>,  // populated by run()
    }

    #[pymethods]
    impl Pipeline {
        #[new]
        #[pyo3(signature = (*, remote, streams, keys=None, key_prefix=None, mode="historical", token=None, since=None))]
        fn new(
            remote: String,
            streams: Vec<String>,
            keys: Option<Vec<String>>,
            key_prefix: Option<String>,
            mode: &str,
            token: Option<String>,
            since: Option<u64>,
        ) -> PyResult<Self> {
            if streams.is_empty() {
                return Err(PyValueError::new_err("streams must be non-empty"));
            }
            if keys.is_some() && key_prefix.is_some() {
                return Err(PyValueError::new_err("keys and key_prefix are mutually exclusive"));
            }
            let mode = match mode {
                "historical" => Mode::Historical,
                "streaming" => Mode::Streaming,
                other => return Err(PyValueError::new_err(format!("mode must be 'historical' or 'streaming', got {other:?}"))),
            };
            Ok(Self { config: PipelineConfig { remote, streams, keys, key_prefix, mode, token, since }, session: None })
        }

        fn run(&mut self, py: Python) -> PyResult<()> {
            if matches!(self.config.mode, Mode::Streaming) {
                return Err(PyNotImplementedError::new_err("streaming mode ships in Phase 31"));
            }
            // Resolve token: arg > env > None
            let effective_token = self.config.token.clone()
                .or_else(|| std::env::var("TALLY_TOKEN").ok());
            let scope = tally::client::Scope::new(
                self.config.streams.clone(),
                self.config.keys.clone(),
                self.config.key_prefix.clone(),
            );
            let cfg = tally::client::ClientConfig {
                remote: self.config.remote.clone(),
                token: effective_token,
                scope,
                mode: tally::client::Mode::Historical,
                since: self.config.since,
            };

            let session = py.allow_threads(|| -> Result<tally::client::Session, tally::client::ClientError> {
                let mut s = tally::client::Session::new(cfg)?;
                s.run()?;
                Ok(s)
            }).map_err(|e| crate::errors::map_client_error(py, e))?;

            self.session = Some(session);
            Ok(())
        }

        fn get(&self, key: &str, stream: &str) -> PyResult<Option<PyObject>> {
            let session = self.session.as_ref().ok_or_else(||
                PyRuntimeError::new_err(".run() must be called before .get()"))?;
            match session.state_store().get(stream, key) {
                Ok(None) => Ok(None),  // null-collapse consistent with v0 query model
                Ok(Some(value)) => Python::with_gil(|py| Ok(Some(value_to_pyobject(py, value)?))),
                Err(tally::client::StateError::OutOfScope) => Err(OutOfScopeError::new_err(format!("({stream}, {key}) is out of declared scope"))),
                Err(e) => Err(crate::errors::map_state_error(e)),
            }
        }

        fn inspect(&self) -> PyResult<HashMap<String, usize>> {
            let session = self.session.as_ref().ok_or_else(||
                PyRuntimeError::new_err(".run() must be called before .inspect()"))?;
            Ok(session.state_store().inspect())
        }
    }
    ```
