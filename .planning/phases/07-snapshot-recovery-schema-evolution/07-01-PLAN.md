<interfaces>
Snapshot header layout (little-endian):
```
| field           | size   | covered_by        |
|-----------------|--------|-------------------|
| magic           | 8 B    | header_crc        |  (b"BEAVASNP")
| format_version  | u16    | header_crc        |  (= 1 for all v0)
| flags           | u16    | header_crc        |  (reserved, = 0)
| created_at_ms   | i64    | header_crc        |
| snapshot_lsn    | u64    | header_crc        |  (last WAL LSN covered)
| registry_version| u64    | header_crc        |
| body_len        | u64    | header_crc        |
| body_crc32c     | u32    | header_crc        |  (crc32c of the body_len bytes that follow)
| header_crc32c   | u32    | (self)            |  (crc32c of bytes [0..44])
```
Fixed size: 8+2+2+8+8+8+8+4+4 = 52 bytes. Truth statement says 48 — let's use 52 actual bytes. (A constant `HEADER_SIZE_BYTES = 52` lives in the module.)

Errors:
```rust
#[derive(Debug, thiserror::Error)]
pub enum SnapshotError {
    #[error("io: {0}")] Io(#[from] std::io::Error),
    #[error("bad magic: expected BEAVASNP, got {got:?}")] BadMagic { got: [u8; 8] },
    #[error("unsupported snapshot format_version {0}")] UnsupportedVersion(u16),
    #[error("header crc mismatch: expected {expected:#010x}, got {got:#010x}")] HeaderCrcMismatch { expected: u32, got: u32 },
    #[error("body crc mismatch: expected {expected:#010x}, got {got:#010x}")] BodyCrcMismatch { expected: u32, got: u32 },
    #[error("truncated snapshot: expected {expected} body bytes, got {got}")] Truncated { expected: u64, got: u64 },
}
```
Returned through `PersistError::Snapshot(SnapshotError)` — add one variant to the existing `PersistError`.

Public API:
```rust
pub const SNAPSHOT_MAGIC: [u8; 8] = *b"BEAVASNP";
pub const SNAPSHOT_FORMAT_VERSION: u16 = 1;
pub const SNAPSHOT_HEADER_SIZE: usize = 52;
pub const SNAPSHOT_EXT: &str = "bvs";

#[derive(Debug, Clone, Copy)]
pub struct SnapshotHeader {
    pub format_version: u16,
    pub flags: u16,
    pub created_at_ms: i64,
    pub snapshot_lsn: Lsn,
    pub registry_version: u64,
    pub body_len: u64,
    pub body_crc32c: u32,
}

impl SnapshotHeader {
    pub fn encode(&self) -> [u8; SNAPSHOT_HEADER_SIZE];
    pub fn decode(bytes: &[u8; SNAPSHOT_HEADER_SIZE]) -> Result<Self, SnapshotError>;
}

pub struct SnapshotWriter;
impl SnapshotWriter {
    /// Atomically writes a snapshot file. Returns the final path.
    /// Write order: tmp file → write header (with body_crc computed first) →
    /// write body → fsync file → rename to .bvs → fsync directory.
    pub fn write(
        dir: &Path,
        snapshot_lsn: Lsn,
        registry_version: u64,
        body: &[u8],
    ) -> Result<PathBuf, PersistError>;
}

pub struct SnapshotReader;
impl SnapshotReader {
    /// Opens, fully verifies (header CRC + body CRC), and returns (header, body_bytes).
    /// Body is returned in memory — caller is responsible for bincode-decoding.
    pub fn open(path: &Path) -> Result<(SnapshotHeader, Vec<u8>), PersistError>;
}

/// Return snapshots in directory sorted by snapshot_lsn DESCENDING.
/// File naming: `snapshot-{snapshot_lsn:016x}.bvs`.
pub fn list_snapshots(dir: &Path) -> Result<Vec<(Lsn, PathBuf)>, PersistError>;

/// Delete all but the `keep` highest-LSN snapshots.
/// Returns count removed.
pub fn prune_old_snapshots(dir: &Path, keep: usize) -> Result<u32, PersistError>;
```

Atomic write protocol (tmp name: `snapshot-{snapshot_lsn:016x}.tmp`, final: `snapshot-{snapshot_lsn:016x}.bvs`):
1. Open tmp for write (truncate if existing tmp from a crashed prior write).
2. Write header bytes, write body bytes.
3. `file.sync_all()` — fsync the file contents.
4. `std::fs::rename(tmp, final)`.
5. Open parent dir and `File::sync_all()` on it (cannot `sync_all` dirs on all platforms; use nix / libc on unix; on Windows sync not needed for rename durability). On macOS + Linux: open dir with `std::fs::File::open(dir)`, call `.sync_all()`. Accept that non-unix targets skip this step.
