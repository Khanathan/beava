### T4 — agg_compile parsing + Rule 11 validation
**RED:** Tests in `agg_compile.rs`: each new op kind parses; `half_life` required for decay ops; invalid half_life rejected with new ErrorCode `AggregationInvalidHalfLife`; `sub_window` required for `burst_count`; `sigma` defaults to 3.0 for `outlier_count`.
**GREEN:** Extend `parse_agg_kind` for 16 ops. Extract `half_life`, `sub_window`, `sigma` params. Add validation. Add ErrorCode variant.

### T5 — WindowedOp extension (TWA custom fold + simple folds)
