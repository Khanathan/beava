```rust
//! Append-only SSD event log with per-stream files.
//!
//! Events are written to per-stream log files using BufWriter<File>.
//! Writes are buffered (BufWriter::write_all is ~100-300ns memcpy).
//! fsync is done periodically via a background timer, never on the hot path.
//! Compaction rewrites log files excluding entries older than history_ttl.

use std::fs::{File, OpenOptions};
use std::io::{BufReader, BufWriter, Read as IoRead, Write};
use std::path::{Path, PathBuf};
use std::time::{Duration, SystemTime};
use ahash::AHashMap;
use serde::{Serialize, Deserialize};

/// Default history TTL: 72 hours (3 days) per CONTEXT.md locked decision.
pub const DEFAULT_HISTORY_TTL: Duration = Duration::from_secs(72 * 3600);

/// A single log entry: timestamp + raw event payload bytes.
#[derive(Debug, Serialize, Deserialize)]
pub struct LogEntry {
    pub timestamp: SystemTime,
    pub payload: Vec<u8>,
}

pub struct EventLog {
    log_dir: PathBuf,
    writers: AHashMap<String, BufWriter<File>>,
    /// Per-stream history TTL for compaction. Streams not in this map are not logged.
    history_ttls: AHashMap<String, Duration>,
}
```

3. Implement EventLog methods:

`pub fn new(log_dir: PathBuf) -> std::io::Result<Self>` -- creates log_dir if needed, returns empty EventLog.

`pub fn register_stream(&mut self, stream_name: &str, history_ttl: Option<Duration>) -> std::io::Result<()>` -- sanitizes stream_name for filesystem safety (replace `/`, `\`, `..`, NUL with `_`), stores history_ttl (using DEFAULT_HISTORY_TTL if None), opens/creates `{log_dir}/{sanitized_name}.log` in append mode, wraps in BufWriter, inserts into writers map. If writer already exists for this stream, do nothing (idempotent re-registration).

`pub fn append(&mut self, stream_name: &str, event_bytes: &[u8], now: SystemTime) -> std::io::Result<bool>` -- looks up writer for stream_name. If not found, returns Ok(false). Creates LogEntry { timestamp: now, payload: event_bytes.to_vec() }. Serializes with postcard::to_stdvec. Writes [u32 BE encoded.len()][encoded bytes] to the BufWriter. Returns Ok(true).

`pub fn read_entries(&self, stream_name: &str) -> std::io::Result<Vec<LogEntry>>` -- opens the log file for reading (separate from the writer). Reads sequential [u32 len][bytes] entries, deserializes each with postcard::from_bytes. Returns all entries. If file does not exist, returns empty vec.

`pub fn fsync_all(&mut self) -> std::io::Result<()>` -- for each writer: flush() then get_ref().sync_data() (fdatasync). This is called from background timer only, never on the hot path.

`pub fn compact_stream(&mut self, stream_name: &str, now: SystemTime) -> std::io::Result<usize>` -- reads all entries from the stream's log file. Filters out entries whose timestamp is older than history_ttl from now. Writes surviving entries to `{stream_name}.log.tmp`. Flushes and closes the old writer first. Renames tmp to replace the original. Reopens BufWriter for the stream. Returns count of removed entries. Uses cooperative yielding concept (though actual yield_now is for the async integration in Task 2).

`pub fn deregister_stream(&mut self, stream_name: &str) -> std::io::Result<()>` -- flushes the writer, removes it from the writers map. Does NOT delete the log file (preserve history for potential re-registration).

`pub fn get_history_ttl(&self, stream_name: &str) -> Option<Duration>` -- returns the history_ttl for a stream.

`pub fn registered_streams(&self) -> impl Iterator<Item = &str>` -- returns iterator over registered stream names.

`fn sanitize_stream_name(name: &str) -> String` -- private helper. Replaces `/`, `\`, NUL bytes with `_`. Replaces `..` with `__`. Returns sanitized string. This prevents path traversal (T-06-04).

7. Add fsync background timer (1-second interval, matching Redis everysec pattern):
```rust
let fsync_state = state.clone();
tokio::spawn(async move {
    let mut interval = tokio::time::interval(Duration::from_secs(1));
    interval.tick().await; // skip first
    loop {
        interval.tick().await;
        // Flush and fdatasync all event log writers
        // Clone file descriptors or access under brief lock
        let result = {
            let mut app = fsync_state.lock().unwrap_or_else(|e| e.into_inner());
            if let Some(ref mut log) = app.event_log {
                log.fsync_all()
            } else {
                Ok(())
            }
        };
        if let Err(e) = result {
            eprintln!("Event log fsync failed: {}", e);
        }
    }
});
```

Note: fsync_all is called under the lock because BufWriter::flush + sync_data are fast for typical buffer sizes. The lock hold is brief (< 1ms). For truly large flushes, this could be moved to spawn_blocking, but that optimization is unnecessary until profiling shows contention.

8. Add compaction background timer (60-second interval per CONTEXT.md locked decision):
```rust
let compact_state = state.clone();
tokio::spawn(async move {
    let mut interval = tokio::time::interval(Duration::from_secs(60));
    interval.tick().await; // skip first
    loop {
        interval.tick().await;
        let now = SystemTime::now();
        // Get list of streams to compact
        let streams_to_compact: Vec<String> = {
            let app = compact_state.lock().unwrap_or_else(|e| e.into_inner());
            if let Some(ref log) = app.event_log {
                log.registered_streams().map(String::from).collect()
            } else {
                vec![]
            }
        };
        // Compact each stream (re-acquires lock per stream for cooperative yielding)
        for stream_name in &streams_to_compact {
            {
                let mut app = compact_state.lock().unwrap_or_else(|e| e.into_inner());
                if let Some(ref mut log) = app.event_log {
                    match log.compact_stream(stream_name, now) {
                        Ok(removed) if removed > 0 => {
                            eprintln!("Compacted {}: removed {} expired entries", stream_name, removed);
                        }
                        Err(e) => {
                            eprintln!("Compaction failed for {}: {}", stream_name, e);
                        }
                        _ => {}
                    }
                }
            }
            // Yield between streams for cooperative scheduling
            tokio::task::yield_now().await;
        }
    }
});
```
