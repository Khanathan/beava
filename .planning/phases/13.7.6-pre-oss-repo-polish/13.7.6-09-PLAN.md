Per filter-repo docs, the callback is a Python EXPRESSION/BODY (not a module) executed against `commit` whose `message` attribute is **bytes** (filter-repo serialises commit messages as `bytes`, not `str`). The body MUST decode → operate on str → re-encode → assign back as bytes. Implementation:
      ```python
      # Phase 13.7.6-09 trailer-strip callback per D-02 (trailer-only surgical strip).
      # `commit.message` is BYTES (filter-repo convention). Decode → operate on str → re-encode.
      # Removes:
      #   - Co-Authored-By: Claude ... lines (any line starting with 'Co-Authored-By:' that contains 'Claude' or 'noreply@anthropic.com')
      #   - "🤖 Generated with [Claude Code]" lines
      #   - Orphaned trailing blank lines created by the above removals
      # Preserves:
      #   - Every other line of the message; commit subjects mentioning Claude survive (D-02 rationale)

      msg = commit.message.decode('utf-8')
      lines = msg.splitlines()
      out = []
      for line in lines:
          stripped = line.strip()
          stripped_lower = stripped.lower()
          # Drop Co-Authored-By trailers that mention Claude / Anthropic noreply
          if stripped_lower.startswith('co-authored-by:') and (
              'claude' in stripped_lower
              or 'noreply@anthropic.com' in stripped_lower
          ):
              continue
          # Drop the "🤖 Generated with [Claude Code]" markers and a few common variants
          if stripped.startswith('🤖 Generated with') and 'Claude' in stripped:
              continue
          if stripped.startswith('Generated with [Claude Code]'):
              continue
          out.append(line)
      # Trim orphaned trailing blank lines
      while out and out[-1].strip() == '':
          out.pop()
      commit.message = ('\n'.join(out) + '\n').encode('utf-8')
      ```
    Write this script as a standalone Python file in the artifacts dir; Plan 11 invokes it via `--commit-callback "$(cat artifacts/13.7.6-09-trailer-strip.py)"`.
