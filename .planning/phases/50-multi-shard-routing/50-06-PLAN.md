Create a helper fn `check_shard_key_fields` (in a suitable module, e.g. `src/server/routing.rs`
or inline in tcp.rs/http.rs):

```rust
/// Check that all fields declared in the stream's tuple shard_key are present
/// in the event payload. Returns the list of missing field names (empty = OK).
pub fn check_shard_key_fields(
    stream_def: &StreamDefinition,
    event_fields: &std::collections::HashMap<String, serde_json::Value>,
) -> Vec<String> {
    match &stream_def.shard_key {
        None | Some(ShardKeySpec::Single(_)) => vec![], // Single field missing handled by primary key heuristic
        Some(ShardKeySpec::Tuple(fields)) => fields
            .iter()
            .filter(|f| !event_fields.contains_key(f.as_str()))
            .cloned()
            .collect(),
    }
}
```
