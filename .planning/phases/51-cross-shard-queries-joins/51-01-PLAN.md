<objective>
Implement lazy global watermark publish — the core of TPC-PERF-06.

Purpose: Each shard accumulates per-stream events_since_publish counters and publishes its max event_time to a shared atomic array once every N events (default 1024). The global watermark for any stream is derived as min across all per-shard atomics, enabling `GET /streams/{name}` to expose a cross-shard watermark without locking any shard on the hot path.

Output:
- `src/shard/global_watermark.rs` — GlobalWatermarkStore wrapping Arc<Box<[AtomicU64]>> indexed by (shard_id × stream_ord), config reader for BEAVA_WATERMARK_PUBLISH_INTERVAL
- `src/shard/watermark.rs` — WatermarkState extended with events_since_publish: AHashMap<String, u64> and publish_if_due method
- Tests (TDD): property test for publish cadence, global-min invariant test
</objective>

pub struct GlobalWatermarkConfig {
    pub publish_interval: u64,   // clamped 64..=65536, default 1024
}

impl GlobalWatermarkConfig {
    pub fn from_env() -> Self;   // reads BEAVA_WATERMARK_PUBLISH_INTERVAL
}

// Indexed as flat array: slot = shard_id * stream_capacity + stream_ord
// stream_ord is registered at stream-registration time (stable ordinal).
pub struct GlobalWatermarkStore {
    slots: Arc<Box<[AtomicU64]>>,
    n_shards: usize,
    stream_capacity: usize,
    // stream_name -> ordinal (registered at stream-registration time)
    stream_ord: AHashMap<String, usize>,
}

impl GlobalWatermarkStore {
    // Called by shard on publish_if_due rollover:
    pub fn publish(&self, shard_id: usize, stream: &str, max_event_time_ns: u64);

    // Called by scatter-gather / GET /streams reader:
    pub fn global_min(&self, stream: &str) -> Option<u64>;   // min across all shard slots

    // Register a stream ordinal (called at REGISTER time, before any shard publish).
    pub fn register_stream(&mut self, stream: &str) -> usize;
}

       - `GlobalWatermarkConfig::from_env()` reads BEAVA_WATERMARK_PUBLISH_INTERVAL, parses u64, clamps to 64..=65536, defaults to 1024 on parse failure or absence.
       - `GlobalWatermarkStore` flat AtomicU64 array indexed by (shard_id * stream_capacity + stream_ord). Use `AtomicU64::new(0)` for uninitialized slots — 0 means "no publish yet" so global_min skips slots where value is 0 (stream not yet observed on that shard).
