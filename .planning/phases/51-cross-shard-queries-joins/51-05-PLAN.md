    ```

    ```

    ```
    curl -s http://localhost:8080/debug/shards | jq '{shard_count, ready, hot_shards_count: (.hot_shards | length)}'
    ```

    Expected: {"shard_count": 2, "ready": true, "hot_shards_count": 0} (or a number if keys are imbalanced).

    `"join operator between '{A}' and '{B}' requires matching shard_key; got '{keyA}' vs '{keyB}'. Fix: declare @bv.stream(shard_key='{suggested_common}') on both streams."`

    Restart with BEAVA_WATERMARK_PUBLISH_INTERVAL=64 and confirm startup INFO log contains "watermark_publish_interval=64".
