## Design notes

- **`fixture()` defaults `test_mode=True`** per Phase 13.5 D-05 cross-amendment from 13.4 D-03. Verified by `test_fixture_default_test_mode_true` introspecting the signature.
- **`replay()` uses `_event` discriminator key** — same convention as the demo datasets generated in Plan 06.
- **`assert_features_eq` uses `math.isclose`** for any value where either side is a `float` (sketch / decay / EWMA accumulators are not bitwise stable). `rel_tol=1e-9, abs_tol=1e-12` defaults.
- **`MockApp` mirrors `bv.App`'s 7-method surface** — register, push, get, batch_get, reset, ping, close. Canned responses via `_set_get_response(table, key, response)`. Unknown `(table, key)` returns `{}` per the cold-start contract in `docs/sdk-api/python.md`.
- **Composite-list keys normalized to tuple** in MockApp's lookup to keep the dict-key hashable.
