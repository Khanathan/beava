## Risks & Mitigations

| Risk | Mitigation |
|---|---|
| A "v0-migrated" skipped test un-skips to red because the feature has a subtle gap | Reclassify as PORT; fix the test, not the engine — engine work is out-of-scope for this phase. If the engine truly lacks the feature, escalate to 26-04 for sign-off discussion. |
| Deleting `_dataframe.py` breaks a live import path | Read the file first; grep for every importer before deletion; only delete if zero importers outside the triaged tests. |
| Test count drops below 744 after deletions | Count survivors + new tests from the v0 milestone; if below floor, pause and reconcile with the SUMMARY (may indicate over-aggressive deletion). |
| Grep false negatives (e.g., `FeatureSet` as a local variable name in unrelated code) | Spot-check the grep output manually; only remove genuine imports and decorator usages. |
