<interfaces>
From python/tally/_dataframe.py (being removed):
- Stream(name, key): creates a streaming source
- Table(name, key): creates a table source
- GroupBy(stream, key): groups a stream by key
- JoinedTable(left, right): joins two streams
- Dataset(name, ...): defines a complete pipeline

From python/tally/_expr.py (being removed):
- Column(name): column reference (df["col"])
- Expr: expression tree node
- EventProxy: _event.field access
These are ONLY used by _dataframe.py. The server-side expression evaluator uses string expressions from tl.derive().

From python/tally/_dataset.py (replacement):
- @dataset(depends_on=[...])
- group_by("key").agg(**ops)
- tl.derive("expr")
- tl.union(a, b)
</interfaces>
</context>
