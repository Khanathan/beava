### Rule 3 blocking fixes

**1. [Rule 3 - Blocking] Fixed tests/profile_ingest.rs closure type-inference errors + enabled pprof feature in samply script**
- **Found during:** Task 1 samply probe invocation (first attempt).
- **Issue:** `tests/profile_ingest.rs` had 7 compile errors: `E0282: type annotations needed` on closure parameters iterating pprof `Frames::frames`. Upstream pprof-crate API tightening means `.first()` returns an ambiguously-typed reference; `leaf.name()` and `sym.name()` couldn't resolve. Additionally, the script `scripts/samply-probe-json-share.sh` did not pass `--features pprof-endpoint` — pprof is an optional dep (`Cargo.toml:96`), so even with the type fix, the test wouldn't compile without the feature.
- **Root cause:** Upstream pprof-crate version drift (no explicit version bump in our `Cargo.toml` since the crate was added); the test worked at 59.6 Wave 7 on a slightly older resolved version.
- **Fix:** (1) Rewrote both closure blocks in `tests/profile_ingest.rs` (lines 271-287 + 412-418) to destructure `frames.frames.first()` and `frame.first()` into explicit `if let Some(frame) = ... / if let Some(leaf) = frame.first()` sequences with explicit `&pprof::Symbol` type annotations on the leaf + sym bindings. (2) Added `--features pprof-endpoint` to the `cargo test` invocation in `scripts/samply-probe-json-share.sh`.
- **Files modified:** `tests/profile_ingest.rs`, `scripts/samply-probe-json-share.sh`.
- **Commit:** ed4037e.
