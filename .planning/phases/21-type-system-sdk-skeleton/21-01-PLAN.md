<objective>
Establish the v0 SDK foundation: the `Stream` and `Table` runtime types, their decorator class-form for declaring external sources, schema declaration via class attributes + type hints, the `tl.col()` expression builder, and `.describe()` for introspection. Delete the old `@tl.source` / `@tl.dataset` / `EventSet` / `FeatureSet` / `_dataframe` surface and the tests that target it.

Purpose: Phases 22/23/25/26 all depend on being able to *declare* a Stream or Table with a known inferred schema and reference columns via `tl.col`. This plan lands the foundation so subsequent plans in Phase 21 (stateless ops + aggregation/join stubs) have a real type to attach operator methods to.

Output: Working `@tl.stream` / `@tl.table` class-form decorators, a `tl.col` expression DSL, `.describe()` introspection, and full unit test coverage of happy paths + surgical error messages. No Rust changes.
</objective>
