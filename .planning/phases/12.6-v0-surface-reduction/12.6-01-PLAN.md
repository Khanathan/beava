<objective>
Drop-in rewrite of `TestServer` to spawn `ServerV18` (mio data plane) instead of the legacy `Server`, while preserving the public builder API surface so the ~20 test files that already use `TestServer::spawn()` / `TestServer::builder()` recompile with no functional changes. Fix the deterministic `phase11_smoke::all_eleven_ops_round_trip_through_http` failure inline (D-02 — set assertion, not key-specific), since the test exercises the rewritten harness end-to-end and HashMap iteration nondeterminism in the `type_mix` Map response is the documented root cause.

Purpose: This plan is the keystone for Phase 12.6. Every later plan that deletes legacy axum code (`push.rs`, `http.rs`, `http_admin.rs`, legacy `Server` struct in `server.rs`, runtime_core_glue async `dispatch_wire_request`) requires that `TestServer` no longer depends on those symbols. Landing this first means subsequent kill-the-axum work can compile cleanly without test-suite churn.

- `crates/beava-server/src/testing.rs` rewritten internally (ServerV18 backing) with same public API
- `crates/beava-server/tests/phase11_smoke.rs:235` assertion rewritten to set-membership

</objective>

```rust
pub struct ServerV18 { /* private fields */ }

impl ServerV18 {
    /// Three OS-allocated SocketAddrs: HTTP event-plane, TCP event-plane, admin.
    pub async fn bind(
        http_addr: SocketAddr,
        tcp_addr: SocketAddr,
        admin_addr: SocketAddr,
    ) -> Result<Self, ServerError>;

    pub fn http_addr(&self) -> SocketAddr;
    pub fn tcp_addr(&self) -> SocketAddr;
    pub fn admin_addr(&self) -> SocketAddr;

    /// `serve` runs the mio event loop until the shutdown future resolves.
    pub async fn serve<F: Future<Output = ()>>(self, shutdown: F) -> Result<(), ServerError>;

    /// Same as `serve` but allows passing wal_dir / snapshot_dir overrides.
    pub async fn serve_with_dirs<F: Future<Output = ()>>(
        self,
        wal_dir: PathBuf,
        snapshot_dir: PathBuf,
        snapshot_interval_ms: u64,
        wal_fsync_interval_ms: u64,
        shutdown: F,
    ) -> Result<(), ServerError>;

    /// Cleanly stops the event loop and joins all worker threads.
    pub async fn shutdown(self);

    /// Accessor for white-box tests: returns shared AppState Arc.
    pub fn app_state(&self) -> Arc<crate::AppState>;

    /// Accessor for white-box tests: returns shared registry Arc.
    pub fn registry(&self) -> Arc<beava_core::registry::Registry>;

    /// Manual snapshot trigger (Phase 7 Plan 03 → ServerV18 equivalent).
    pub fn snapshot_trigger_handle(&self) -> crate::snapshot_task::SnapshotTriggerTx;
}
```
