## Wire schema bump posture

| Option | Description | Selected |
|--------|-------------|----------|
| Hard cutover — refuse with error (Recommended) | Push payload schema deletes `event_time_ms` cleanly; if a client sends it, server returns 400 with structured `unknown_field` error. Matches Redis-shaped commitment. | |

**Notes:** Locked as **D-03**. v0 isn't released → no users to break → no need for any compat layer. Simplifies Phase 12.6 substantially: no deprecation window, no parse-and-strip path, no recovery compat shim.
