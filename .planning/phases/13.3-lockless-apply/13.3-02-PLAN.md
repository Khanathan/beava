<objective>
Cut the entire server — every write AND read handler — over from the Mutex-based
`DevAggState` to a unified `Rc<RefCell<AppState>>` shared via axum `State<_>`.
After this plan, the word "lock" does not appear on any apply-path state access.
No channel, no actor, no mpsc. Direct borrow, direct mutation, direct response.

Purpose: unlock the throughput projected in CONTEXT SC3 (≥ 5× lift on at least
one cell). The Mutex was the bound; removing it is the lift.

Output: `AppState` struct + every handler rewritten + DevAggState retired.
Workspace tests green, RYW preserved, fsync discipline preserved.
</objective>

```rust
pub struct AppState {
    // Non-atomic: guarded by the outer Rc<RefCell<AppState>>.
    pub state_tables: BTreeMap<String, AggStateTable>,
    pub temporal_stores: HashMap<String, TemporalStore>,
    pub event_id_index: HashMap<u64, EventIdEntry>,
    pub wal_buffer: InMemWalBuffer,

    // Atomics: live directly on AppState (outside the RefCell-guarded collections).
    // Accessed lock-free from handlers via `state.borrow().counters.next_event_id.fetch_*`.
    pub counters: Counters,

    // Arc — Registry has its own orthogonal RwLock.
    pub registry: Arc<Registry>,
}

pub struct Counters {
    pub next_event_id: AtomicU64,
    pub max_event_time_ms: AtomicU64,
}
```

Handler signature pattern:
```rust
pub async fn push_handler(
    State(state): State<Rc<RefCell<AppState>>>,
    Path(event_name): Path<String>,
    Json(body): Json<Value>,
) -> impl IntoResponse {
    // ... pre-borrow work: parse row, resolve event_time_ms ...

    let (ack_lsn, fsync_rx) = {
        let mut s = state.borrow_mut();
        let event_id = s.counters.next_event_id.fetch_add(1, Relaxed);
        s.counters.max_event_time_ms.fetch_max(event_time_ms as u64, Relaxed);
        let lsn = s.wal_buffer.append_inmem(&record);
        s.apply_event(&event_name, row, event_time_ms, event_id);
        let fsync_rx = if is_sync { Some(fsync_worker_tx.request(lsn)) } else { None };
        (lsn, fsync_rx)
    }; // borrow released HERE — before any .await

    if let Some(rx) = fsync_rx { rx.await?; }
    Ok(Json(json!({ "event_id": ack_lsn })))
}
```
