| surface_id | surface_name | proposed_test_path | test_function_name | rationale |
|---|---|---|---|---|

| SRV-REG-03-evolve | schema-evolution diff matrix | `python/tests/v0/test_schema_evolution.py` | `test_additive_succeeds`, `test_destructive_returns_409`, `test_changed_field_type_returns_409` | Three subcases per SRV-REG-03: (a) adding a new aggregation feature → 200 + version bump; (b) removing an existing feature without `force=True` → 409 + `force_required` error; (c) changing a feature's window → 409.
