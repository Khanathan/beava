<objective>
Implement the `tally reshard` offline migration CLI tool (TPC-DX-03, CONTEXT D-06/D-07).
The tool reads the source data-dir snapshot + per-shard logs, rehashes every key via
`hash(event.key) mod K`, writes a new v8 snapshot and per-shard logs to out_dir, and
optionally replaces the original data-dir atomically via `--replace`.

Purpose: Operators who need to scale from N=1 to N=K shards (or any Nâ†’K) can migrate their
live data dir offline in a single command. Downtime equals tool runtime. The original data
dir is never modified without explicit `--replace`. CONTEXT D-06: binary is `tally reshard`;
D-07: rehash logic lives in a shared `reshard` module reused by the CLI and in-process path.

Output: `src/reshard/` module with `rehash_to_shard` and `reshard_data_dir`, plus the
`tally reshard` subcommand wired in `src/main.rs`. TDD test suite in
`tests/test_reshard_cli.rs`.
</objective>
