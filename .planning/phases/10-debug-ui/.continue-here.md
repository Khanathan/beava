## Critical Anti-Patterns (do NOT repeat these)

| Pattern | Severity | Prevention |
|---------|----------|-----------|
| Adding a 16px typography token or a 12px spacing token to the UI-SPEC | advisory | UI-SPEC §3.2 and §2.1 explicitly prohibit them. ui-checker will BLOCK. If you need 16px for a metric, use 14px mono. If you need 12px of padding, use 8px or 16px. |
| Double-counting throughput on cascade/fan-out | advisory | Dedupe with `HashSet<String>` per push in the throughput tracker. Unit test required (listed in RESEARCH §Validation Architecture). |
| Forgetting `list_views()` when building /debug/topology nodes | advisory | Views live in a different map than streams. Research Pitfall 7. |
| Adding a new /debug/memory/per_stream endpoint instead of extending /debug/memory | advisory | Extend additively per CONTEXT.md and RESEARCH §Pitfall 8. |
| Using older axum path syntax `*file` or `:file` | advisory | axum 0.8 requires `{*file}`. Existing codebase already uses brace syntax. |
| Committing d3 v7 + dagre-d3 0.6.4 without a browser smoke test | advisory | dagre-d3 is maintenance mode (last release 2020). Smoke test rendering before committing vendored files; fall back to d3 v5 if it fails. (HIGH-risk assumption A3 in RESEARCH.) |
