**Criterion 6 is the star test.** Implementation:

```python
def test_c6_identical_registry_state_across_transports(beava_binary):
    # Spawn two servers so registry state starts fresh for each
    from beava._embed import spawn_embedded_server, teardown_process
    descriptors = [TransactionEvent, UserProfileTable, CheckoutDerivation]

    # Round 1: via http://
    proc_a, http_url_a, _ = spawn_embedded_server()
    try:
        with bv.App(http_url_a) as app:
            app.register(*descriptors)
        state_a = httpx.get(f"{http_url_a}/registry", headers={}).json()
    finally:
        teardown_process(proc_a)

    # Round 2: via tcp://
    proc_b, http_url_b, tcp_url_b = spawn_embedded_server()
    try:
        with bv.App(tcp_url_b) as app:
            app.register(*descriptors)
        # GET /registry is HTTP-only; use http_url of the second subprocess
        state_b = httpx.get(f"{http_url_b}/registry").json()
    finally:
        teardown_process(proc_b)

    # Strip `_dev_only` and normalize any non-deterministic fields
    for s in (state_a, state_b):
        s.pop("_dev_only", None)
    assert state_a == state_b, f"registry state diverges:\\n{state_a}\\nvs\\n{state_b}"
```

Note on GET /registry: only available when `BEAVA_DEV_ENDPOINTS=1` on the subprocess. Plan 03-04's `spawn_embedded_server` sets this by default; the `beava_server` fixture already sets it too.

**Python README.md skeleton:**

```markdown
# beava — Python SDK

Real-time feature server, Python SDK.

## Install

    pip install beava

## Quickstart — HTTP

    import beava as bv

    @bv.event
    class Transaction:
        user_id: str
        amount: float
        event_time: int

    @bv.table(key="user_id")
    class UserProfile:
        user_id: str
        balance: float

    with bv.App("http://localhost:7379") as app:
        resp = app.register(Transaction, UserProfile)
        print("registry_version:", resp["registry_version"])

## Quickstart — TCP (faster path)

    with bv.App("tcp://localhost:7380") as app:
        app.register(Transaction, UserProfile)

## Quickstart — Embed mode (no separate server)

    with bv.App() as app:
        app.register(Transaction, UserProfile)

Embed mode auto-spawns a local `beava` binary on ephemeral ports. Requires
the binary to be on `PATH` (`brew install beava` or `docker pull beava/beava`)
or set `BEAVA_BINARY=/path/to/beava`.

## Validate without network I/O

    errs = bv.App().validate(Transaction, UserProfile)
    assert errs == []

`validate` returns `list[beava.ValidationError]` — empty on success.
```
