Side-by-side comparison · Gaegu operator ≈ in middle
pandas (batch)
events.groupby("user_id").agg(
    views_total=("path", "count"),
    last_seen=("ts", "max"),
)
≈
beava (live)
events.groupby("user_id").agg(
    views_total=bv.count(),
    last_seen=bv.latest(e.ts),
)
Same shape, different time model. pandas runs once over a frame. beava maintains the groupby forever.
