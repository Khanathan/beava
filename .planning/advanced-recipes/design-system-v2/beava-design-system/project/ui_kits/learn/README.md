# Learn (field guide) UI Kit — beava.dev/learn

Blog-style multi-chapter field guide. Uses the same tokens and shares components with the docs kit where possible.

## Components
- `LearnNav.jsx` — site nav
- `LearnHero.jsx` — chapter hero with illustration
- `LearnArticle.jsx` — long-form prose with serif headings and callouts
- `ChapterGrid.jsx` — index grid of chapters (at the bottom)

`index.html` is Chapter 3: "Detecting fraud with rolling counters."
