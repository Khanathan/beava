---
title: "beava"
description: "Real-time feature server for fraud, ad-tech, and behavioral analytics. Single binary, in-memory state, declarative pipeline."
---

beava is a single-binary real-time feature server. Push events over HTTP,
declare aggregations, query features by entity key — sub-millisecond
reads, no Kafka, no Spark, no JVM.
