# Tally Logo & Mascot — AI Image Generation Prompts

## About the Product

Tally (codename: Streamlet) is a real-time feature server — think "Redis with a built-in streaming aggregation engine." You push events in over a binary TCP protocol and get computed features back instantly. One Rust binary, zero infrastructure, sub-millisecond latency. Built for fraud detection, ML feature serving, and real-time AI agent context. The tagline: **Push events, get features.**

## Modifier Stacks

Mix and match these add-on phrases with any variant above to adjust the output:

**More goofiness:**
> exaggerated cartoon proportions, squash and stretch animation style, googly eyes, oversized head, tiny limbs, rubberhose animation influenced

**More professionalism:**
> clean vector logo, suitable for business card printing, minimal unnecessary detail, balanced composition, corporate-friendly while retaining character

**Icon / Favicon:**
> simple silhouette version, recognizable at 16x16 pixels, monochrome variant, reduce to essential shapes only, works as a tiny app icon

**Dark mode:**
> designed to work on dark backgrounds, bright saturated accent colors, subtle glow or neon effects on key elements, light-colored outlines instead of black

**Style references:**
> in the style of beloved tech company mascots like ScyllaDB's monster, DigitalOcean's Sammy the shark, Docker's Moby Dock whale, the Go Gopher, and Rust's Ferris the crab — fun and recognizable characters that represent serious technical products
